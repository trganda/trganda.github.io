        长亭雷池

        Aug 14, 20233 min read

        计划安装长亭雷池社区版本，部署在其它 web 服务之前。

        安装

        安装方式很简单，这里直接使用安装脚本

        安装位置为

        /opt/safeline

        可修改 /opt/safeline/.env 文件的来更改端口，默认监听中 9443，使用 https

        MGT_PORT=9443

        修改端口至 2053 的目的是为了使用 cloudflare 的代理服务，2053 属于支持的 https 端口范畴。否则直接访问的话，会报 https 证书不安全，目前还不知道如何为雷池配置证书。

        修改其它容器的配置

        重新修改了 gitea 的配置，相比 搭建 Gitea 并备份 Obsidain 增加 nginx 作为反向代理，仅绑定至 127.0.0.1:80，同时在 nginx.conf 配置文件中设置白名单只允许本地 ip 访问 gitea 服务。

        这样做的目的是避免能够从外部直接访问到服务，否则雷池就白装了。

        配置

        在防护站点中配置希望被防护的网站

        以前是直接将域名 tea.trganda.top 指向 gitea 服务的，现在让它指向雷池防火墙

        在编辑站点中设置对外的域名和端口

        可以选择上传对应域名的证书，这里透露就直接自动生成了，另外再设置上游服务器，也就是真正的服务。

        参考

        1. https://waf-ce.chaitin.cn/docs/guide/install 『在线安装』
        2. https://waf-ce.chaitin.cn/docs/faq/config 『站点配置』
