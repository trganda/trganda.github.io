        该利用链多用于验证反序列化漏洞的存在，通过它能够发起 DNS 请求。调用过程如下

        Gadget Chain:
  HashMap.readObject()
    HashMap.putVal()
      HashMap.hash()
        URL.hashCode()

        这个利用链的最终目的是调用能够发起 DNS 请求的方法。
