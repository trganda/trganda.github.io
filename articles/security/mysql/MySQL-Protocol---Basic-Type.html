        Integer Types

        数值类型分为两种，固定长度和可变长度。

        FixedLengthInteger

        固定长度有 6 种，长度分别为 1、2、3、4、6、8 字节。

        TypeLength
        int<1>1 byte
        int<2>2 byte
        int<3>3 bytes
        int<4>4 byte
        int<6>6 byte
        int<8>8 byte

        LengthEncodedInteger

        可变长度遵循以下规则

        • [0, 251)：int<1>
        • [251, 2^16)：0xFC + int<2>
        • [2^16, 2^24)：0xFD + int<3>
        • [2^24, 2^64)：0xFE + int<8>

        String Types

        一共 5 种

        FixedLengthString

        固定长度字符串，例如 ERR_Package 中的 State 字段，长度固定为 5 个字节。

        NullTerminatedString

        以 00 结尾的字符串。

        VariableLengthString

        可变长度的字符串，字符串的长度由其它字段提供，或在运行时进行计算。

        LengthEncodedStrings

        带长度前缀的字符串，前缀说明后续字符串内容的长度。

        RestOfPacketString

        如果字符串处于 Packet 的结尾，那么它的长度可以通过整个 Packet 的长度减去前面的内容的长度而计算出。
