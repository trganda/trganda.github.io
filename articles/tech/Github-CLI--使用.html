        指定请求头

        以 https://api.github.com/users/{username}/starred 为例，该接口会返回指定用户 starred 的仓库列表，可通过 -H 指定请求头。

        指定请求方法和参数

        该接口有一个 GET 参数 per_page，指定每次返回仓库的个数，最大为 100，可通过 -X 指定请求方法，-f 指定请求参数『如果为 POST 请求，参数会放在请求体中』。

        如果需要请求更多的内容，比如所有 starred 的参考『个数超过 100』，则还可以使用 --paginate 参数，表示请求全部内容，数据会以多个数组的形式返回。
